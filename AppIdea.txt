We are going to build a new project called as DocuSmith.
It's an AI Report generator. But it not only generates report but also replaces content in a already built report with new content and it has this another feature called as proposal drafter, where people can chat with our AI SuDOCU and discuss with it to make stunning new proposals. So it has 3 core features:
* Replace Content
* Report Generator
* Proposal Drafter.
These are the 3 core funtions.
This is figma devmode link where I designed the UI:
https://www.figma.com/design/JktV85jKiJshMAvwCjoOKM/DocuSmith?node-id=0-1&m=dev&t=05ZiKHCCNILUEdWp-1
How these core funtions work:
	Replace Content: Users can upload a already built report into the upload content box and press analyze. Our AI will analyze the content and identify the headings sub-headings and contents from it. The next step requires the user to upload some reference to the reference box and if he wants to add any additional heading. Basically after the analysis the AI will provide the table of contents and the user can add more headings and subheadings if he wants to. Plus each of these heading and subheading boxes have another box beside them called no. of words. So when the user upload the new reference like code or abstract of the project(.txt, .jsx, .tsx, .js, .html, .css, .py...) the AI will analyze the reference and no .of words and generate new content and place them under appropriate headings and sub headings according to the reference and almost equal to the no. of words. We will also let the user export the report in different formats 
	Report Generator: The Users can create the structure of format using add headings and sub headings buttons. Basically we let the user create the structure of the format by letting them enter the headings and subheading. Then the user needs to upload the reference code or abstracts so that the AI can generate content according to the project and make the project accurate. After uploading, the user has to just press submit an let AI do the work and make the user a fully fledged report. It should also make the table of contents and main pages like a fully fledged report. We will also let the user export the report in different formats.
	Proposal Drafter: Users can chat with our AI just like chatgpt, Gemini and other chatbots and discuss with the user so that an stunning proposal can be made. Just give the chatbot idea of the project and let AI do the work. We will also let the user export the proposal in different formats.
These are the core features of the project.
